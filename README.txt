This is a social calendar for family application. This can application will sync the family's events by using the google calendar.

Feature:
1. User can add events detail and can invite people by using email address. After the user add the event, the event information will sync to family's calendar
2. User can delete an event.
3. After an event has been added, the text to void will be utilised.
4. This app will read the event list in particular day.

Library:
1. SQLite3

Framework:
1. CVCalendar
2. FMDB
3. EventKit
4. EventUIKit

Framworks that are installed by cocoapod:
1. Alamorfire
2. SwiftyJSON
3. gtm-oauth2
